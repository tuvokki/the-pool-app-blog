This is a tutorial sample project that might evolve into something useful.

How to use this.
----------------
Install play! and make sure the play command works nicely in your path (see http://www.playframework.org/documentation/1.2.3/install)
Checkout this repository and cd into the main dir
	cd ~/poolratings
	play run

To develop further on this app open it in Textmate or Eclipse (or any etitor at all)
	play eclipsify
	(in eclipse) Import -> Existing project ... etc.
